JS13k
=====

Let's Play!

Requirements
---

NodeJS 6.x+

Install dependencies:
---
With npm:

        $ npm install

Development
---

1. Run the compiler in watch mode

        $ npm start

2. Now you can open the app at dist/index.html.

Production
---

1. Build the zip file:

        $ npm run build

2. ...

3. Profit! (submit dist/game.zip to js13k)

Clean
---

To cleanup build artifacts:

        $ npm run clean
